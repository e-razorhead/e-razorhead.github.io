# Personal GitHub Page

This is my personal GitHub Pages site, built with Jekyll.

## Local Development

To run this site locally:

1. Install Jekyll and Bundler:
   ```bash
   gem install jekyll bundler
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Run the Jekyll server:
   ```bash
   bundle exec jekyll serve
   ```

4. Open your browser and navigate to `http://localhost:4000`

## Theme

Currently using the **minima** theme. You can change the theme in `_config.yml`.

## GitHub Pages

This site is automatically built and deployed by GitHub Pages when you push to the `main` branch.
